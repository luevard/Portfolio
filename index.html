Mes Projets

Allin - Pari entre amis

Ce projet consiste en la réalisation d’un jeu mobile de paris entre amis qui se joue en misant des jetons fictifs sur un pari qu’un joueur écrit. Il prend la forme de deux clients, l’un pour les mobiles Android et l’autre pour les iOS. Ce projet inclut également un Web-Service en Ktor qui possède, en interne, une base de données. Les clients sont développés uniquement en natif (IOS/ANDROID).

Composition de services

Ce projet est une simulation de création de prêt bancaire à travers différentes API REST développées respectivement en PHP et JAVA. Un client en Guzzle est aussi disponible afin de manipuler les demandes de prêts.

PocketBook

Ce projet est la création d'une application multi-plateforme MAUI en reprenant l'affichage de l'application Musique sur IOS tout en mettant en place le patron d'architecture MMVM.

MMIX

MMIX est un projet IA permettant de prédire le gagnant d'un combat de MMA entre 2 combattants, ce projet s'appuie sur une classification en Random Forest. Le projet a été développé en Python avec la librairie SKLEARN.
